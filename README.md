# ChipLab VLSI - Verilog HDL Basic Digital Design

This repository contains fundamental digital design modules implemented in Verilog HDL as part of the ChipLab Academy VLSI training program. The designs focus on combinational logic circuits and are intended to build a strong foundation in RTL design and digital systems.


📌 Modules Included

1. Half Adder
- Implements basic addition of two 1-bit inputs
- Outputs: Sum and Carry

2. Full Adder
- Adds three 1-bit inputs (A, B, Carry-in)
- Outputs Sum and Carry-out

3. Ripple Carry Adder
- Multi-bit binary adder built using full adders
- Demonstrates carry propagation across stages

 4. Full Subtractor (Structural)
- Performs binary subtraction using structural modeling
- Outputs Difference and Borrow

5. 1-bit Comparator
- Compares two 1-bit inputs
- Outputs logic for Equal, Greater, and Less conditions


🛠️ Tools Used
- Verilog HDL
- RTL Design Principles
- Digital Logic Design Concepts
- Simulation tool (e.g., ModelSim / Vivado / Synopsys VCS if applicable)


Learning Outcomes
- Understanding combinational logic design
- RTL modeling using Verilog
- Structural and behavioral design approaches
- Basics of modular digital system design
